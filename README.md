# AI-Based Sales and Inventory Analysis Dashboard

## 📌 Project Overview
This project is an AI-based sales and inventory analysis dashboard developed for El Yoser, a company specialized in the manufacturing and sale of paint buckets.

The goal of this project is to help the company analyze sales data, monitor inventory, and support decision-making through interactive visualizations and intelligent insights.

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- Streamlit  
- Matplotlib / Plotly  

---

## 🚀 Features
- 📊 Interactive dashboard for sales analysis  
- 📈 Visualization of monthly sales trends  
- 🏆 Identification of top-selling products  
- 🎛️ Data filtering by product, region, and date  
- 📦 Inventory monitoring and low stock detection  
- 🤖 Rule-based AI insights and recommendations  

---

## 🧠 AI Component
The system includes a rule-based AI component that generates insights such as:
- Detecting high-demand products  
- Identifying low stock situations  
- Providing recommendations for better decision-making  

---

## 📊 Data Processing
The application loads data from a CSV file, cleans and processes it using Pandas, and calculates key performance indicators (KPIs) such as:
- Total sales  
- Average sales  
- Product performance
